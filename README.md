# Turtle Crossing Game

A Frogger-style arcade game built using Python Turtle Graphics and Object-Oriented Programming principles.

## Features

- Dynamic car generation with random spawning
- Real-time collision detection
- Progressive difficulty increase with level advancement
- Keyboard-based player controls
- Score and level tracking system
- Smooth game loop and screen updates

## Technologies Used

- Python
- Turtle Graphics
- Object-Oriented Programming (OOP)

## Project Structure

- `main.py` → Main game loop and event handling
- `player.py` → Player movement and finish-line logic
- `car_manager.py` → Car creation, movement, and speed control
- `scoreboard.py` → Level tracking and game-over display

## Controls

| Key | Action |
|-----|--------|
| Up Arrow | Move Forward |
| Down Arrow | Move Backward |

## Concepts Practiced

- OOP Design
- Collision Detection
- Event Handling
- Game Loop Management
